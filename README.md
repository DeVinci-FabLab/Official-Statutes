# Statuts Officiels — DeVinci Fablab

Ce dépôt contient les statuts officiels de l'association **DeVinci Fablab**, régie par la loi du 1er juillet 1901, dont le siège social est au Pôle Universitaire Léonard de Vinci (PULV), Courbevoie.

## Structure des statuts

- **Article 1** — Définition : forme, objet, siège, durée, dissolution, ressources, règlement intérieur
- **Article 2** — Membres : fondateurs, d'honneur, émérites, annuels, perte de qualité, persona non grata
- **Article 3** — Comités : Contrôle, Direction (Présidence, Trésorerie, Secrétariat), Gestion, Soutien, Bureau
- **Article 4** — Assemblée Générale : ordinaire et extraordinaire
